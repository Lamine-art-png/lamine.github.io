# AGRO-AI Enterprise Portal Walkthrough Script

Use this script for customer, integration partner, and investor walkthroughs of `https://app.agroai-pilot.com`.

## Five minute interactive flow

1. **Open the portal** at `https://app.agroai-pilot.com`.
   - Position AGRO-AI as the Water Command Center: recommendation, execution, and verification for connected irrigation environments.
2. **Launch the workspace** with the launch workspace button.
   - Explain that the walkthrough uses isolated simulated telemetry, visible through the discreet **Mode: Demo** pill and dismissible telemetry banner.
3. **Orient the audience on Command Center**.
   - Use **How to read this workspace** to explain the connected controller environment, block-level recommendation, scheduled/applied water, field observation, verification, and report.
4. **Select a scenario** on Command Center.
   - Use Dry day, Rain forecast, Partial telemetry, Planned vs applied mismatch, or Verification completed successfully depending on the audience.
5. **Generate a recommendation**.
   - Open **Intelligence** and click **Generate recommendation**.
   - Show decision, timing, depth, duration, confidence, data quality, key drivers, source trace summary, live inputs, manual overrides, execution task, and verification requirement.
6. **Schedule irrigation**.
   - Click **Schedule recommendation** and show the Scheduled step update.
7. **Confirm applied water**.
   - Click **Mark as applied** and show the Applied step update with evidence.
8. **Add field observation**.
   - Click **Add observation** and show the Observed step update.
9. **Verify outcome**.
   - Click **Verify outcome** and explain that AGRO-AI tracks the complete recommendation → scheduled → applied → observed → verified chain.
10. **Generate report**.
   - Click **Open report** or **Preview Report** in Reports.
   - Show the report preview, then demonstrate **Export CSV** or **Print Report**.
11. **Show live WiseConn recommendation**.
   - Use Customer Access / live status preview, open Intelligence, optionally enter overrides, and click **Run Live WiseConn Recommendation**.
   - If the API fails, explain that the portal surfaces a customer-safe error and keeps demo mode available.

## 30 second version

1. Launch the workspace.
2. Note the single **Mode: Demo** pill and dismissible simulated-telemetry banner.
3. Show **Alpha Vineyard · Command Center** and the passive **Operating Journey** strip.
4. Use the action bar to schedule, mark applied, add observation, verify outcome, and open the report.
5. Show Integrations and explain WiseConn/Talgil runtime status and backend credential requirements.

## Talk track

AGRO-AI tells growers what to irrigate, shows what was scheduled, verifies what was actually applied, and makes the result visible in one place.

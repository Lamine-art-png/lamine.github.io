Platform API / Changelog

Changelog

We ship additively. New endpoints and response fields can arrive within v1; breaking changes ship under a new version. Write clients that tolerate unknown fields.

2026-07-21 Docs

Unified developer platform

Launched the standalone AGRO-AI developer platform: a new landing experience, restructured documentation, a spec-driven API reference, and an in-browser API Explorer.

2026-07-10 Added

Soil-water balance endpoint

Added GET /v1/fields/{field_id}/water-balance returning root-zone depletion, readily-available water, and a stress-risk signal.

2026-06-28 Improved

Signed webhook deliveries

Webhook deliveries now include an AGROAI-Signature header (HMAC-SHA256) and are retried with backoff for up to 24 hours. See Webhooks.

2026-06-12 Added

Model selection for evapotranspiration

The ET endpoint accepts a model query parameter (openet_ensemble, ssebop, eemetric), defaulting to the ensemble.

2026-05-30 Beta

Platform API v1

Introduced fields, evapotranspiration, weather, and irrigation recommendations with sandbox and production keys.

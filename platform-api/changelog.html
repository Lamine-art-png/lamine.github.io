Platform API / Changelog

Changelog

The Platform API is in private beta. It ships additively within v1: new endpoints and response fields can arrive; write clients that tolerate unknown fields. Any change to the public contract is reviewed and digest-pinned.

2026-07-21Docs

Developer platform, generated from the contract

Introduced this unified developer platform. The API reference is generated directly from the curated backend contract, and a CI check fails the build if the docs drift from the server or if any private route would leak.

2026-07Added

Usage & request logs

Added GET /v1/platform/usage and GET /v1/platform/request-logs for metered usage and request history.

2026-07Improved

Provider readiness surfaced in the contract

Provider readiness is now expressed in the contract. EarthDaily and Valley remain awaiting partner contracts, and Valley physical writes stay disabled.

2026-07Beta

Platform API private beta

Curated surface for providers, fields, sources, observations, recommendations, reports, jobs, usage, and request logs — with project-scoped agro_test_ / agro_live_ keys issued from the console.

One API for the whole water picture

Composable, well-documented resources that map to how agronomists actually reason about water.

Fields

Register fields by GeoJSON boundary or centroid. Everything else hangs off a field.

Evapotranspiration

Daily ETc from a satellite-driven model ensemble — the backbone of any water budget.

Soil-water balance

Root-zone depletion and stress risk, updated daily, so you know when water matters.

Irrigation recommendations

Model-backed depth and timing advice — advisory only, never actuating hardware.

Weather

bash
curl -X POST https://api.agroai-pilot.com/v1/irrigation/recommendations \
  -H "Authorization: Bearer $AGROAI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{ "field_id": "fld_3nZq7Kp2", "horizon_days": 7 }'
python
import os, requests

resp = requests.post(
    "https://api.agroai-pilot.com/v1/irrigation/recommendations",
    headers={"Authorization": f"Bearer {os.environ['AGROAI_API_KEY']}"},
    json={"field_id": "fld_3nZq7Kp2", "horizon_days": 7},
    timeout=30,
)
print(resp.json()["recommended_depth_mm"])
javascript
const res = await fetch(
  "https://api.agroai-pilot.com/v1/irrigation/recommendations",
  {
    method: "POST",
    headers: {
      Authorization: `Bearer ${process.env.AGROAI_API_KEY}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({ field_id: "fld_3nZq7Kp2", horizon_days: 7 }),
  }
);
const rec = await res.json();
console.log(rec.recommended_depth_mm);

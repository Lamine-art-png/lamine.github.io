Security and reliability

Scoped machine access, not shared browser credentials.

Service accounts are isolated by organization, project, environment, workspace, provider, resource, scope, and optional CIDR policy. Secrets are shown once, webhooks use signed durable delivery, and every administrative decision is audited.

Test

Synthetic fixtures, resettable projects, no customer data, no physical execution.

Live

Review-gated access, production readiness checks, explicit entitlements and limits.

Webhooks

Versioned events, HMAC signatures, durable retries, delivery history and redelivery.

Providers

Truthful capability status. EarthDaily and Valley remain awaiting partner contracts.

SDK foundations

Typed clients for Python and TypeScript.

Use pagination helpers, typed errors, safe retries, idempotency keys, uploads, job polling, usage access, and webhook signature verification without building the protocol layer yourself.

Support

We want you shipping. Here's how to get unblocked and reach the team.

Before you reach out

  • Check the API reference for exact parameters and responses.
  • Reproduce the call in the API Explorer against sandbox data.
  • Review Errors — the error.code usually names the fix.
  • Capture the request_id from the error body; it lets us find the exact request.

Contact

Email developers@agroai-pilot.com with your workspace, the endpoint, and a request_id. For account, billing, and key management, use the developer portal.

Security issue? Email security@agroai-pilot.com and please don't file it publicly.

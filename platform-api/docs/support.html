Support

The Platform API is in private beta. Here's how to get unblocked.

Before you reach out

  • Check the API reference for exact parameters, scopes, and schemas.
  • Review Errors — the code usually names the fix, and a 404 can mean the feature isn't enabled for your org.
  • Capture the request_id from the response so we can find the exact request.

Contact

Use the console for account, project, and key management. For API questions during the beta, email developers@agroai-pilot.com with your organization, the endpoint, and a request_id.

●

Security issue? Email security@agroai-pilot.com and please don't file it publicly.

Base URL & versioning

All requests go to a versioned base URL. The current version is v1.

EnvironmentBase URL
Productionhttps://api.agroai-pilot.com/v1
Sandboxhttps://api-preview.agroai-pilot.com/v1

We version the API in the path. Backwards-incompatible changes ship under a new version; additive changes (new endpoints, new response fields) can arrive within v1, so write clients that tolerate unknown fields.

Conventions

  • Resource-oriented REST with predictable, plural nouns (/fields).
  • JSON request and response bodies; UTF-8; snake_case field names.
  • Every object carries an id and an object type.
  • Timestamps are Unix epoch seconds unless the field name ends in _at (ISO 8601).
  • Lists use cursor pagination.

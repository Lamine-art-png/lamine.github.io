Quickstart

Authenticate, register a field, and read your first water-intelligence result. About five minutes.

1. Get a sandbox key

Create a workspace in the developer portal and copy a sandbox key from Settings → API keys. Sandbox keys are prefixed sk_test_ and run against realistic sample data — no card, no live billing.

Export it so the examples below can read it from your environment:

bash
export AGROAI_API_KEY="sk_test_your_sandbox_key"

Keep keys server-side. Never embed an API key in a browser bundle, mobile app, or public repository. See Authentication.

2. Register a field

Everything in AGRO-AI hangs off a field. Create one with a name, crop, and boundary:

bash
curl -X POST https://api-preview.agroai-pilot.com/v1/fields \
  -H "Authorization: Bearer $AGROAI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "North Quarter — Almonds",
    "crop": "almond",
    "centroid": { "lat": 36.331, "lon": -119.292 }
  }'

The response includes the field id you will use in later calls:

201 · application/json
{
  "id": "fld_3nZq7Kp2",
  "object": "field",
  "name": "North Quarter — Almonds",
  "crop": "almond",
  "area_hectares": 32.4,
  "created": 1752710400
}

3. Read water intelligence

Ask for a model-backed irrigation recommendation for that field:

bash
curl -X POST https://api-preview.agroai-pilot.com/v1/irrigation/recommendations \
  -H "Authorization: Bearer $AGROAI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{ "field_id": "fld_3nZq7Kp2", "horizon_days": 7 }'
200 · application/json
{
  "object": "irrigation_recommendation",
  "field_id": "fld_3nZq7Kp2",
  "recommended_depth_mm": 24.0,
  "recommended_window": { "start": "2026-07-21", "end": "2026-07-22" },
  "confidence": 0.82,
  "rationale": "Root-zone depletion approaching RAW with 7.4 mm/day ETc and no forecast precipitation."
}

Recommendations are advisory. The API never actuates irrigation hardware — you stay in control of what happens in the field.

4. Do it in code

python
import os, requests

BASE = "https://api-preview.agroai-pilot.com/v1"
headers = {"Authorization": f"Bearer {os.environ['AGROAI_API_KEY']}"}

rec = requests.post(
    f"{BASE}/irrigation/recommendations",
    headers=headers,
    json={"field_id": "fld_3nZq7Kp2", "horizon_days": 7},
    timeout=30,
).json()

print(f"Apply {rec['recommended_depth_mm']} mm ({rec['confidence']:.0%} confidence)")
javascript
const BASE = "https://api-preview.agroai-pilot.com/v1";
const res = await fetch(`${BASE}/irrigation/recommendations`, {
  method: "POST",
  headers: {
    Authorization: `Bearer ${process.env.AGROAI_API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({ field_id: "fld_3nZq7Kp2", horizon_days: 7 }),
});
const rec = await res.json();
console.log(`Apply ${rec.recommended_depth_mm} mm`);

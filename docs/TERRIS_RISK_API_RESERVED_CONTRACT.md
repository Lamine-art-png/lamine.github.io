# Terris Risk API Reserved Contract

Terris Risk API is reserved for future institutional portfolio risk summaries. It is disabled by default and no public endpoint is introduced in this pass.

Future contract types may include:
- portfolio summary scope
- field and farm exposure references
- risk signal type and severity
- evidence packet references
- truth-label summary
- missing-input summary
- generated timestamp and provenance

Current limitations:
- no fake portfolio metrics
- no public API route
- no production readiness claim
- no compliance filing claim

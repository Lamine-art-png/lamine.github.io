# Strategic partner integration program

Partner applications capture technical and commercial qualification without
placing credentials or documents in generic JSON. Documents use source-library
or object-storage references; credentials use the versioned encrypted vault.

The administrative dossier records owners, NDA/contract status, documentation,
authentication, sandbox, endpoint allowlist, schemas, rate limits, webhooks,
retention, support contacts, milestones, readiness, blockers, custom rate card,
and custom limits. Every administrative change is audited.

EarthDaily and Valley Irrigation remain `awaiting_partner_contract`. No endpoint,
authentication mechanism, schema, limit, or live capability is invented.
Valley write readiness can only remain `disabled` or `contract_review`; physical
write execution remains disabled independently.

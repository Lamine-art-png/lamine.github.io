# Platform API terms review

The system stores versioned acceptance for API Terms, Acceptable Use, Privacy,
and an optional Data Processing Addendum. Records include user, organization,
document version, timestamp, privacy-safe IP hash, bounded user agent, and audit
event. A document may require reacceptance.

No legal text in this branch is represented as attorney-approved. Before terms
enforcement can be enabled, counsel must approve the exact versions, retention,
regional applicability, age/authority representation, acceptable-use remedies,
privacy disclosures, DPA flow, and reacceptance policy.
